# mv-interactive
A comfortable mv command ❤️

```shell
npm install @aloof/mv-interactive
```

<img src="https://raw.githubusercontent.com/sean-codes/mv-interactive/main/example.gif" />

# Usage

```
$ mvi

> Move file promt shows

Arrows = Navigate
Space = Select File
Enter = Confirm

> Confirm choose [yes, no, copy, delete]

Move complete!
```
